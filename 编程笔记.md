# python

## 基础

- 力口的判题策略是相同类生命一次，但是方法调用多次。因此使用全局变量需要重新初始化
- 全局变量在函数中使用需要global，否则相当于重新定义变量
- Python 中调用类函数需要使用self.前缀，并在函数参数中传入self
- 由于python中没有设置return的参数类型。所以`return visited[state]==1` 不能不要==1，否则返回的就是1(int)而不是true(bool)
- python中位操作和c++相同

## np
